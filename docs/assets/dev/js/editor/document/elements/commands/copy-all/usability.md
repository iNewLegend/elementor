# Command
  `$e.run('document/elements/copy-all')`

### User Action / Area of effect
     Copy all elements.
     
### Effect
    Copy all elements to browser stroage.

## Output
   * **Fails**: Throws error on fail.
   
## How to delete element?
```javascript
$e.run( 'document/elements/copy-all' );
```
